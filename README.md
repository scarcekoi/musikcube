## Usage

1. Make sure this directory exists: `~/.config/musikcube/` (e.g.: `mkdir -p ~/.config/musikcube/themes/`).
2. Download the flavor(s) of your choice from [themes](themes).
3. Move the file(s) to `~/.config/musikcube/`.
4. Open Musikcube and disable **degrade to 256 color palette** in the settings.
5. In the settings, under **color theme**, select the theme you would like.
6. Enjoy!
